# Проект "sending letters"
## Введение в проект (Описание проекта)
"sending letters" - это "консольная" программа для отправки писем через библиотеку smptb lib


## Цели проекта
отправкa писем через библиотеку
для установки зависимости в первую очередь должен быть скачен "python 3".
"python 3" рекомендуется скачать с официального сайта:
```
https://www.python.org/downloads/
```
## Запуск проекта и работа с ним
### Функционал проекта
возможность отправки писем на почты yandex
### Начало работы
В первую очередь необходимо заменить шаблоны и нажать на кнопку запустить



Проект выполнен в образовательных целях на онлайн-курсе "Основы Python" школы "Лидер".
