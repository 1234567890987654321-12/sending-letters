# Проект "sending letters"

## Описание проекта
Проект "sending letters" позволяет быстро сформировать и отправить маркетинговое письмо

## Используемые технологии
* Python3
* smtplib - библиотека python для отправки писем

## Инструкция по запуску
1. Подготовьте почту
   1. Включите доступ для сторонних приложений в настройках почты.
   2. Создайте специальный пароль приложения
2. Откройте терминал и запустите скрипт вставив в него следующую команду
   ```
   python main.py
   ```

3. Заполните содержимое письма
   * После запуска программа последовательно запросит:
      1. Данные для письма
      2. Название вашей команды
      3. Данные отправителя и получателя
      4. Логин и пароль приложения
